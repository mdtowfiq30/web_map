About This Web Map

This web map is created for learning purposes to explore how web GIS works. The map integrates multiple layers such as boundaries, divisions, districts, and more, demonstrating how geographic data can be visualized and interacted with on the web.

The map uses technologies like Leaflet.js to add functionality, allowing users to explore different layers, interact with geospatial data, and download specific data layers for further use.

This project is aimed at understanding the fundamentals of web-based geographic information systems (GIS) and their applications.
